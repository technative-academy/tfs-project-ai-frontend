# 2BasedGames - AI-Powered Pun intended game site

**2BasedGames** is a frontend project created as part of **TechNative Bootcamp Challenge 5** by **Abdu and Warren**.
This is an AI powered platform where users can get **Game Pun titles** built with **HTML, CSS, and JavaScript**.

---

## Technologies used

**HTML** - for the semantic structure of the pages
**CSS** - for the styling and layout
**JavaScript** - for DOM rendering, API request and interactivity like tabs
**APIs** - for pun generated titles for games and product listings
**GitHub** - Collaboration and version control
**GitHub pages** - for hosting the site
